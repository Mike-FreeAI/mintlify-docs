## Prerequisites

<Info>
  You need Node.js version 19 or higher installed on your system before proceeding with the setup.
</Info>

## Installation and Setup

### Installing Mintlify CLI

Follow these steps to get Mintlify running on your local machine:

<Steps>
  <Step title="Install the CLI tool">
    Choose your preferred package manager to install Mintlify globally:

    <CodeGroup>
      ```bash npm
      npm i -g mintlify
      ```

      ```bash yarn
      yarn global add mintlify
      ```
    </CodeGroup>
  </Step>

  <Step title="Start local development">
    Navigate to your documentation directory (where the `mint.json` file is located) and run:

    ```bash
    mintlify dev
    ```
    
    This will start a local preview server at `http://localhost:3000`.
  </Step>
</Steps>

### Port Configuration

By default, Mintlify runs on port 3000. You have two options for port configuration:

1. **Custom Port**: Use the `--port` flag to specify a different port:
    ```bash
    mintlify dev --port 3333
    ```

2. **Automatic Port Selection**: If port 3000 is busy, Mintlify will automatically find the next available port:
    ```md
    Port 3000 is already in use. Trying 3001 instead.
    ```

## Version Management

### Updating Mintlify CLI

Each CLI release corresponds to a specific Mintlify version. To ensure compatibility with the production version, keep your CLI updated:

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

## Link Management

### Checking for Broken Links

Use the CLI's built-in link validator to find and fix broken references:

```bash
mintlify broken-links
```

## Common Issues and Solutions

### Node.js Related Issues

<Accordion title="Sharp Module Error on M1/M2 Macs">
  If you see the error "Could not load the 'sharp' module using the darwin-arm64 runtime", follow these steps:

  1. Remove existing Mintlify installation:
     ```bash
     npm remove -g mintlify
     ```
  2. Upgrade Node.js to v19 or higher
  3. Reinstall Mintlify:
     ```bash
     npm install -g mintlify
     ```
</Accordion>

### General Troubleshooting

<Accordion title="Unknown Errors and Cache Issues">
  If you encounter unexpected errors:

  1. Delete the Mintlify cache folder:
     ```bash
     rm -rf ~/.mintlify
     ```
  2. Restart the development server:
     ```bash
     mintlify dev
     ```
</Accordion>
